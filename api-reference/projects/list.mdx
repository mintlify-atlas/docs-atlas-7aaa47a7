---
title: "List Projects"
api: "GET /api/projects"
description: "Retrieve all projects for the authenticated user"
---

Returns a list of all projects belonging to the authenticated user.

## Authentication

Requires Bearer token authentication.

## Response

<ResponseField name="id" type="integer" required>
  The unique identifier for the project
</ResponseField>

<ResponseField name="name" type="string" required>
  The name of the project (max 100 characters)
</ResponseField>

<ResponseField name="color" type="string" required>
  The color code for the project in hex format (e.g., #8b949e)
</ResponseField>

<ResponseField name="userId" type="integer" required>
  The ID of the user who owns the project
</ResponseField>

## Example Request

```bash
curl -X GET https://api.example.com/api/projects \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

## Example Response

```json
[
  {
    "id": 1,
    "name": "Website Redesign",
    "color": "#3b82f6",
    "userId": 123
  },
  {
    "id": 2,
    "name": "Mobile App",
    "color": "#10b981",
    "userId": 123
  }
]
```

## Error Responses

<ResponseField name="401 Unauthorized">
  Authentication token is missing or invalid
</ResponseField>

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```
