---
title: "Reorder Tasks"
api: "PUT /api/tasks/reorder"
description: "Update the position/order of multiple tasks"
---

## Authentication

This endpoint requires authentication. Include a Bearer token in the Authorization header.

## Request Body

Send an array of objects containing task IDs and their new positions.

<ParamField body="items" type="array" required>
  Array of objects with task ID and position information
</ParamField>

<ParamField body="items[].id" type="integer" required>
  The task ID to reorder
</ParamField>

<ParamField body="items[].position" type="integer" required>
  The new position for the task
</ParamField>

## Response

Returns 204 No Content on success.

## Example Request

```bash
curl -X PUT https://api.dailytracker.com/api/tasks/reorder \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '[
    {
      "id": 1,
      "position": 3
    },
    {
      "id": 2,
      "position": 1
    },
    {
      "id": 3,
      "position": 2
    }
  ]'
```

## Example Response

```
204 No Content
```
